# Background Images

Place your background images in this folder:

## Required Images:

### 1. Welcome Page Background
- **File:** `welcome-bg.jpg`
- **Size:** 1920x1080px or larger
- **Format:** JPG, PNG, or WebP
- **Description:** Beautiful church or spiritual background for the welcome page

### 2. Login Page Background
- **File:** `login-bg.jpg`
- **Size:** 1920x1080px or larger
- **Format:** JPG, PNG, or WebP
- **Description:** Professional background for the login page

### 3. Admin Dashboard Background
- **File:** `admin-bg.jpg`
- **Size:** 1920x1080px or larger
- **Format:** JPG, PNG, or WebP
- **Description:** Subtle background for the admin dashboard

## Image Guidelines:

- **High Resolution:** Use images that are at least 1920x1080px
- **Optimized:** Compress images for web use (under 500KB each)
- **Consistent Style:** Use similar color tones across all backgrounds
- **Professional:** Choose images that match your church's aesthetic

## How to Add Images:

1. Save your images in this folder with the exact filenames above
2. The app will automatically use them as backgrounds
3. Images will be responsive and work on all screen sizes

## Fallback:

If no background images are provided, the app will use gradient backgrounds as fallbacks.
